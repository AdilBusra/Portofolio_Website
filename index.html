Do you know HTML, CSS, JavaScript, and Tailwind?

HTML (HyperText Markup Language)

HTML adalah fondasi dari setiap halaman web. Ibaratnya, HTML adalah kerangka atau tulang dari sebuah bangunan. Fungsinya adalah untuk menyusun struktur konten seperti teks, gambar, dan link menggunakan elemen-elemen yang disebut 'tag'.

CSS (Cascading Style Sheets)

Jika HTML adalah kerangkanya, maka CSS adalah arsitek interiornya. CSS digunakan untuk mendesain dan menata tampilan visual halaman web, seperti mengatur warna, font, layout, dan membuat website menjadi responsif di berbagai perangkat.

JavaScript (JS)

JavaScript adalah 'otak' yang membuat website menjadi hidup dan interaktif. Dengan JS, kita bisa membuat fitur seperti animasi, validasi form, slider gambar, dan merespons aksi pengguna. JS-lah yang mengubah halaman statis menjadi aplikasi dinamis.

Tailwind CSS

Tailwind adalah sebuah framework CSS yang modern. Berbeda dengan CSS biasa, Tailwind menyediakan 'utility classes' yang memungkinkan kita mendesain langsung di dalam HTML tanpa menulis file CSS terpisah. Ini sangat mempercepat proses development.

2

Menyusun Struktur dengan HTML

Buat file `tes.html`. Gunakan tag HTML untuk membangun kerangka dasar website. 

<!DOCTYPE html>
<html lang="id">
<head>
  <title>Website Saya</title>
  <link rel="stylesheet" href="tes.css">
</head>
<body>
  <header> ... </header>
  <footer> ... </footer>
  <script src="tes.js"></script>
</body>
</html>
<!DOCTYPE>
Deklarasi untuk memberitahu browser bahwa ini adalah dokumen HTML5.
<html>
Tag utama yang membungkus semua konten halaman web.
<head>
Berisi info meta (judul, link CSS) yang tidak tampil di halaman.
<title>
Judul halaman yang muncul di tab browser.
<link>
Menghubungkan file eksternal seperti CSS.
<body>
Membungkus semua konten yang terlihat oleh pengguna.
<header>
Bagian pembuka yang berisi judul atau navigasi.
<footer>
Bagian penutup halaman, biasanya berisi info hak cipta atau kontak.
<script>
Menghubungkan file JavaScript untuk interaktivitas.
3

Menata Tampilan dengan CSS

Buat file `tes.css` untuk mengatur warna, font, dan layout agar website terlihat menarik. 

/* Di dalam file tes.css */
body {
  font-family: 'Arial', sans-serif;
  background-color: #f4f4f4;
}
header {
  background-color: #333;
  color: #fff;
}
font-family
Menentukan jenis huruf (font) yang digunakan untuk teks.
background-color
Mengatur warna latar belakang sebuah elemen.
color
Mengatur warna teks di dalam sebuah elemen.
text-align
Mengatur perataan horizontal teks (kiri, kanan, tengah).
padding
Memberi ruang di bagian dalam sebuah elemen.
margin
Memberi jarak di bagian luar sebuah elemen.
4

Menambah Interaktivitas dengan JavaScript

Buat file `tes.js` untuk membuat website lebih dinamis. Berikut adalah beberapa contoh fitur interaktif yang bisa ditambahkan menggunakan JavaScript:

Mode Gelap/Terang

Membuat tombol yang bisa mengubah skema warna website, memberikan kenyamanan visual bagi pengguna di malam hari.

Tombol "Back to Top"

Menampilkan tombol yang muncul saat halaman di-scroll ke bawah, yang berfungsi untuk membawa pengguna kembali ke puncak halaman dengan cepat.

Validasi Form

Memeriksa input pengguna di form kontak sebelum dikirim, misalnya memastikan email sudah terisi dengan format yang benar.

Jendela Modal/Popup

Menampilkan jendela popup saat sebuah tombol diklik, berguna untuk menampilkan informasi detail, gambar, atau form login tanpa pindah halaman.

5

Publikasi ke Internet (Deploy)

Setelah website selesai, langkah terakhir adalah mempublikasikannya agar bisa diakses semua orang. Berikut adalah alur proses deploy menggunakan Vercel.

1. Unggah ke GitHub

Simpan seluruh folder proyek Anda ke sebuah repository di akun GitHub Anda.

2. Hubungkan ke Vercel

Buat akun di Vercel.com (gunakan login via GitHub) dan berikan izin akses ke repository Anda.

3. Import & Deploy

Di dashboard Vercel, import repository dari GitHub. Cukup klik tombol "Deploy".

4. Selesai & Bagikan

Vercel akan memberikan link unik yang bisa langsung Anda bagikan ke siapa saja.
